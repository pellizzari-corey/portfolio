---

# Overview

This repository contains a React-based static site that is deployed automatically to AWS using GitHub Actions.

The site is built with **Vite + React + TypeScript** and hosted using **AWS S3 and CloudFront**.

Every push to the `main` branch triggers an automated deployment pipeline.
